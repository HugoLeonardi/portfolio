Réalisations techniques pour le BTS SIO SISR

Présentation

Dans le cadre de ma formation BTS SIO option SISR, j'ai réalisé deux projets professionnels majeurs. Ces projets illustrent ma capacité à gérer une infrastructure, de son déploiement initial jusqu'à sa mise en production.

J'ai souhaité orienter mes réalisations vers les technologies actuelles du marché, en mettant l'accent sur l'automatisation (Infrastructure as Code) et le Cloud Computing, tout en conservant une base solide en administration système classique.

Projet 1

Déploiement Cloud Nextcloud — AWS & Terraform

Déploiement automatisé d'une solution de stockage collaboratif (Nextcloud) sur AWS via Terraform. Architecture trois tiers : serveur web EC2, base de données RDS et stockage objet S3.
